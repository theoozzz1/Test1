	\section*{Justification}
	We went with Apache 2.0 mainly because all the libraries we use (Spring Boot, Next.js, etc.) already use permissive licences, so there are no compatibility issues. It also has a patent clause which gives us a bit more protection than something like MIT. Overall it lets anyone use or build on our code while still requiring them to give credit, which felt like the right balance for a uni project.
